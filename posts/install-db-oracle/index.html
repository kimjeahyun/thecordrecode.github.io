Oracle 19c

보통 특정 프로그램을 설치할 때는 runinstaller를 사용하여 설치를 하게 된다. 그래픽 과 다양한 추가 옵션을 쉽게 설정 할 수 있다.

하지만 때때로 우리가 설치할 때 그래픽 화면을 접근하지 못하는 경우가 있는데 그래픽 화면을 사용하지 않는 방법을 silent installation 이라고 한다.

hardware version checking

1

cat /etc/redhat-release

Create a Directory

오라클을 설치하기 위해서 u01 , u02 폴더를 각각 생성할 것이다.

oracle_home

1

mkdir -pv /u01/app/oracle/product/19.0.0/dbhome_1

datafile

1

mkdir -pv /u02/oradata

create the new groups and users

1
2
3

groupadd -g 2022 oinstall
groupadd -g 2023 dba
useradd oracle -g 2022

change ownership and permissions

1
2

chown -R oracle:oinstall /u01 /u02
chmod -R 775 /u01 /u02

Pre-requisites required for oracle installation

1

yum install -y oracle-database-preinstall-19c

or

1
2

curl -o oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm https://yum.oracle.com/repo/OracleLinux/OL7/latest/x86_64/getPackage/oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm
yum -y localinstall oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

Unzip software

1
2

cd /u01/app/oracle/product/19.0.0/dbhome_1/
unzip LINUX.X64_193000_db_home.zip

Execute pre-installation command

1

./runInstaller -executePrereqs -silent -responseFile /u01/app/oracle/product/19.0.0/dbhome_1/install/response/db_install.rsp

OS 메모리가 16G가 아닐때

1

./runInstaller -ignorePrereq -waitforcompletion -silent -responseFile /u01/app/oracle/product/19.0.0/dbhome_1/install/response/db_install.rsp

Run the below script from the root user after installation

1
2

sh /u01/app/oraInventory/orainstRoot.sh
sh /u01/app/oracle/product/19.0.0/dbhome_1/root.sh

Verify after installation

1
2
3

export ORACLE_HOME=/u01/app/oracle/product/19.0.0/dbhome_1/
export ORACLE_SID=orcl
export PATH=/u01/app/oracle/product/19.0.0/dbhome_1/bin:$PATH

netca -silent -responseFile $ORACLE_HOME/assistants/netca/netca.rsp

lsnrctl status
